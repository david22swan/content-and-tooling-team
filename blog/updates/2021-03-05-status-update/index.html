Supported Module Major Releases

The whole team has been super busy this week working on prepping our modules for major releases. We have done 34+ major releases on our supported modules with only a few outstanding. The release includes the removal of Puppet 5, RHEL 5 family and SLES 11 support. If you are running Puppet 5 please note it is no longer supported and we advise you to upgrade to Puppet 7 as soon as you can.

Docker bug fix

We did a minor release (3.14.0) with a specific bug fix for RedHat 7 and the default docker package (version 1.13.1) that fixes an issue in puppetlabs-docker module with a force rewrite of the /etc/sysconfig/docker-storage file. The fix disables management of this file and uses the default /etc/sysconfig/docker-storage-setup preventing the conflict described in the issue. More details available here MODULES-10898.
